CSS Specificity

What is Specificity ?

Specificity คือ ค่าเฉพาะเจาะจง หากมี CSS Rule ที่ขัดแย้งกันตั้งแต่สองขึ้นไปชี้ไปที่ Element เดียวกัน

Browser จะคำนวณหาว่า CSS Rule ไหนมีค่า Specificity มากกว่าและเลือกไปแสดงผลกับ Element

ค่า Specificity สามารถแบ่งตาม Selector ได้ดังนี้
