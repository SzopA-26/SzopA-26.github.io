CSS Specificity

Specificity คืออะไร ?

Specificity คือ ค่าเฉพาะเจาะจง หากมี CSS Rule ที่ขัดแย้งกันตั้งแต่สองขึ้นไปชี้ไปที่ Element เดียวกัน Browser จะคำนวณหาว่า CSS Rule ไหนมีค่า Specificity มากกว่าและเลือกไปแสดงผลกับ Element

ค่า Specificity สามารถจำแนกได้จาก Selector

  • 1000 - Inline styles 
    Example: <p style="color: red;">Sample text here...</p>
  • 100 - IDs 
    Example: #navbarNav
  • 10 - Classes, attributes and pseudo-classes 
    Example: .classes, [attributes], :hover, :focus
  • 1 - Elements and pseudo-elements 
    Example: h1, div, :before, :after

    • วิธีคำนวณ Specificity

    เริ่มจาก 0 แล้วบวกตามค่า Specificity ของแต่ละ Seletor ข้างต้น เช่น

    A: h1
    B: #content h1
    C: <div id="content"><h1 style="color: #ffffff">Heading<h1><div>

    ค่า Specificity ของ A คือ 1 (หนึ่ง Element)

    ค่า Specificity ของ B คือ 101 (หนึ่ง ID และหนึ่ง Element)

    ค่า Specificity ของ C คือ 1000 (Inline styling)

    1 < 101 < 1000 เนื่องจากค่า Specificity ของ C สูงที่สุด C จึงถูกเลือกและนำไปแสดงผล
